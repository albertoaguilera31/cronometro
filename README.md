# cronometro
App de cronómetro realizada con HTML, CSS, Bootstrap y JavaScript. 

Este repositorio contiene una implementación simple de un cronómetro en JavaScript. El cronómetro muestra las horas, minutos y segundos en un formato de dos dígitos y permite controlar su funcionamiento mediante dos botones: "Iniciar/Pausar" y "Reiniciar".

El cronómetro se puede iniciar y pausar haciendo clic en el botón "Iniciar/Pausar". Una vez iniciado, el contador de tiempo se actualiza cada segundo, mostrando el tiempo transcurrido en la interfaz de usuario.

Además, se proporciona un botón de "Reiniciar" para restablecer el cronómetro a cero y detenerlo. Al reiniciar, el cronómetro mostrará "00:00:00" en la pantalla.

El código utiliza funciones para controlar el intervalo de tiempo y dar formato adecuado a las unidades de tiempo (horas, minutos y segundos). También incorpora eventos de clic en los botones para gestionar las acciones del cronómetro.

Este cronómetro es ideal para proyectos que requieran un temporizador simple y adaptable. Su implementación modular permite su fácil integración y personalización según las necesidades del proyecto.
